## OpenSource Penetration and Security Auditing Software
###Allows  users to focus on Computer Forensics, reverse engineering and vulnerability detection

## Kali is not like Ubuntu distros that are commonly used for everyday tasks, Kali linux is purposeful in that bluetooth and network connections are autodisabled when running the distro

## Penetration Testing - Mock cyberattacks launched to find vulnerabilities in a computer
     ###Basically a tool for ethical hacking

## Ethical Hacking - Usage of hacking tools and techniques to fix security weaknesses than to cause harm
   ### A toolbox for ethical hacking, penetration testing and Network Security Assesments. Can be used Ethically and Unethically.

## Ventoy - Opensource tool to create a multi-iso file bootable drive
    ###Formats a usb to hold multiple distros, instead of needing a usb for a single distro/boot

## How to create a bootable drive with Ventoy?
###Download Install package from official Ventoy Website
###Extract tar.gz file
###In terminal run run sudo ./Ventoy2Disk.sh -i  (Must be in Downloads folder for command to run)
###Paste ISO files into Ventoy USB
###Added Kali, Ubuntu, Ubuntu Cinnamon and Nyx OS

## How to boot Kali onto pc?
###Restart Computer
###Enter Bios one time menu by rapidly pressing f12
###Click on Ventoy USB (Dell latitude computer ting)
###Ventoy Boot menu Appears
###Click on Kali ISO
###Enter Kali in Grub2 and amd 64 live
